> **Data Analyst & BI Developer** — I build end-to-end data solutions, from exploratory analysis and ML modelling to interactive dashboards and production monitoring, with a focus on **HR, Finance & Healthcare** domains.

- 🏥 Latest: **ICU Sepsis Early-Prediction** — PyTorch GRU time-series model with a custom MLOps drift-detection layer
- 🤖 **LLM-augmented analytics** — [SQL Analyst Agent](https://github.com/rubenfm77/sql-analyst-agent): plain-English questions → SQL → auto-chart on a synthetic SME credit portfolio
- 💊 **Bioinformatics**: de novo drug candidate generation with ESM-2 protein embeddings + RDKit scoring
- 📊 **Power BI** practitioner: DAX, PBIP semantic models, multi-page HR & sales reports
